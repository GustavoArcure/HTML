Trabalhando com tags para formúlarios

Elementos para formúlarios são voltados à interação com o usúario. Entrada de dados, seleção de opções, processamento para manipulaçao de dados, entre outros.

Formúlarios no HTML são somente uma Interface.

Para que o úsuario funcione, é necessário combiná-lo com uma Linguagem de Programação.
